행사주제 : 청년, 희망, 도약

슬로건 : 우리들의 청춘 Jump!

행사기간 : 2023. 9.16(토)~9.21(목), 6일간

행사장소 : 서울광장과 서울시청 일대

참여대상 : 서울거주·활동 청년, 청년정책에 관심 있는 시민 누구나

주최 : 서울특별시

‘2023 서울 청년을 위한 일주일’에서 청년들이 필요한 정보와 활력도 얻고 역량도 기를 수 있는 한 주를 만나 보시기를 바랍니다!

‘우리들의 청춘 Jump!’를 슬로건으로 9.16(토)~9.21(목) 6일간 개최되며 서울에 거주하거나 활동하는 청년, 청년정책에 관심 있는 시민들과 함께하는 축제입

니다.

청년정책을 직접 경험해 볼 수 있는 청년정책 체험 박람회부터 강연과 멘토링이 진행되는 청년주간 프로그램까지 청년들의 다양한 고민을 해결해 줄 다채로운

프로그램들을 선보입니다.

서울청년주간이 올해는 ‘서울 청년을 위한 일주일‘이라는 이름으로 여러분들을 찾아옵니다.
